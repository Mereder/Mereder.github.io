四数之和

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

注意：

答案中不可以包含重复的四元组。

示例：

给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

满足要求的四元组集合为：
[
[-1, 0, 0, 1],
[-2, -1, 1, 2],
[-2, 0, 0, 2]
]

思路

跟三数之和思路是一样的，我们复习下：

三数之和中，我们先通过预排序，让数组有序。然后固定一个数，然后移动剩下的两个数（双指针）

即：预排序+固定一个+双指针

对于四数之和，我们采用同样的方案，只不过这里需要更改的是，我们固定两个数

  • 特判，特殊情况直接处理
  • 预排序
  • 固定一个数a
    • 处理相同情况
    • 再固定一个数b
      • 处理b的相同情况
      • 定义双指针
      • 双指针进行移动
        • 4sum == target添加结果
        • 4sum < target: 移动左指针
        • 4sum > target: 移动右指针

Posted on 2019-12-08 Edited on 2019-12-09 In 算法刷题 , Leetcode

K个不同整数的子数组

给定一个正整数数组 A，如果 A 的某个子数组中不同整数的个数恰好为 K，则称 A 的这个连续、不一定独立的子数组为好子数组。

（例如，[1,2,3,1,2] 中有 3 个不同的整数：1，2，以及 3。）

返回 A 中好子数组的数目。

示例 1：

输出：A = [1,2,1,2,3], K = 2
输入：7
解释：恰好由 2 个不同整数组成的子数组：[1,2], [2,1], [1,2], [2,3], [1,2,1], [2,1,2], [1,2,1,2].
示例 2：

输入：A = [1,2,1,3,4], K = 3
输出：3
解释：恰好由 3 个不同整数组成的子数组：[1,2,1,3], [2,1,3], [1,3,4].

提示：

1 <= A.length <= 20000
1 <= A[i] <= A.length
1 <= K <= A.length

思路

方案比较巧妙的办法是：

K个不同整数的子数组 = 用至多K个不同整数的结果 - 至多K-1个不同整数的结果

以【1，2，3】为例，K=3时候，

至多K个包含:[1],[2],[3],[1,2],[2,3],[1,3],[1,2,3]

至多K-1个包含: [1],[2],[3],[1,2],[2,3],[1,3]

做差结果 就是 [1,2,3]

Z字形变换

将一个给定字符串根据给定的行数，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 “LEETCODEISHIRING” 行数为 3 时，排列如下：

L C I R
E T O E S I I G
E D H N
之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比如：”LCIRETOESIIGEDHN”。

请你实现这个将字符串进行指定行数变换的函数：

string convert(string s, int numRows);

示例 1:

输入: s = “LEETCODEISHIRING”, numRows = 3
输出: “LCIRETOESIIGEDHN”
示例 2:

输入: s = “LEETCODEISHIRING”, numRows = 4
输出: “LDREOEIIECIHNTSG”
解释:

思路

这种题上来就先模拟下这个的生成过程，模拟过程中就能找到一定的规律：

  • 先从上下下，再从下向上
  • 总是在 0行或者 n-1行转换反向
  • 不要受Z字形诱导，最后的输出结果就是按行从上到下将字符串拼接

最长回文子串

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

示例 1：

输入: “babad”
输出: “bab”
注意: “aba” 也是一个有效答案。
示例 2：

输入: “cbbd”
输出: “bb”

思路

首先判断一个字符串是不是回文字符串，就是看该字符串是否是中心对称的。

  • 奇数个时: aba对称
  • 偶数个时：aa对称

现在是从一个字符串中，找出最长的回文字符子串，只需要遍历一遍字符串，假设每个位置都是一个回文子串的中心，然后根据是否是回文串来验证，记录长度最大的回文子串。

分析：总的遍历为o(N)，两种对称方式的遍历为o(2N)，总的时间复杂度为o(N^2)

最长有效括号

有效括号

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: “()”
输出: true
示例 2:

输入: “()[]{}”
输出: true
示例 3:

输入: “(]”
输出: false
示例 4:

输入: “([)]”
输出: false
示例 5:

输入: “{[]}”
输出: true

思路

首先说只有一种类型的括号的话，我们的解决方案（这也是Leetcode-22-括号生成题目中的一个步骤）。

如果只有一种括号，我们在遍历字符串的时候，只需要注意两个问题：

  • 遍历过程中 左边个数>=右边个数
  • 遍历结束后 左边个数 == 右边个数

对应这个问题，上述方法就行不通了，因为涉及到三种不同的符号，单考虑一种括号的顺序正确是没用的，还需要考虑到括号之间的交叉，比如:

单个符号的顺序没有问题，但是总的就是不正确的括号。

我们可以看到，如果一整个字符串都是有效的，那么其子字符串肯定也是有效的，那么最小的子问题就是一对相邻括号是匹配的。

删除排序数组中的重复项

给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

示例 1:

给定数组 nums = [1,1,2],

函数应该返回新的长度 2, 并且原数组 nums 的前两个元素被修改为 1, 2。

你不需要考虑数组中超出新长度后面的元素。
示例 2:

给定 nums = [0,0,1,1,1,2,2,3,3,4],

函数应该返回新的长度 5, 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4。

你不需要考虑数组中超出新长度后面的元素。

说明:

为什么返回数值是整数，但输出的答案是数组呢?

请注意，输入数组是以“引用”方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:(引用问题嘛，传值传引用)

思路

虽然是个easy的题，但是一开始确实没有很好的思路，其实每次做数组的题老是不会灵活的使用指针（也就是数组的下标），每次都是只有依次遍历的指针。没有灵活使用

这个题使用两个指针(i慢，j快)

  • i指针用来指示有效数字（即无重复数）
  • j指针用来遍历数组所有数，重复就跳过，不重复就赋值到 i+1

比较好的是数组是排好序的，所以单次遍历就能把重复数都找出来。
