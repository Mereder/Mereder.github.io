Leetcode-189-旋转数组

Posted on 2020-02-06 Edited on 2020-02-07 In 算法刷题 , Leetcode

旋转数组

给定一个数组，将数组中的元素向右移动 k 个位置，其中 k 是非负数。

示例 1:

输入: [1,2,3,4,5,6,7] 和 k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右旋转 1 步: [7,1,2,3,4,5,6]
向右旋转 2 步: [6,7,1,2,3,4,5]
向右旋转 3 步: [5,6,7,1,2,3,4]
示例 2:

输入: [-1,-100,3,99] 和 k = 2
输出: [3,99,-1,-100]
解释:
向右旋转 1 步: [99,-1,-100,3]
向右旋转 2 步: [3,99,-1,-100]
说明:

尽可能想出更多的解决方案，至少有三种不同的方法可以解决这个问题。
要求使用空间复杂度为 O(1) 的 原地 算法。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/rotate-array

思路

因为用python 解题嘛，上来最先考虑的就是python特性解决。

根据题意就是个循环移动过程，将尾部的元素拿出来放到头部

对应可用的list的方法就是pop(),insert()

insert(self, index, object, /)

Insert object before index.

pop(self, index=-1, /)

Remove and return item at index (default last).

Raises IndexError if list is empty or index is out of range.

如果深究时间复杂度的话，python底层实现中list的insert()时间复杂度是o(N) 而pop()时间复杂度是o(1)，并且调用k次，所以时间复杂度为o(kn)

正规解题姿势

可用通过多次旋转不同位置的元素来实现。

总共调用了3次reverse，3次的时间复杂度分别为o(n),o(k),o(n-k)

总的时间复杂度还是o(n)的
